## Console Commands

1. Initialize node index.js init → creates hidden folder .s3git
2. Add file.txt (goes to staging): node index.js add hello.txt → creates copy of the file
3. Commit: node index.js commit "initial commit" → makes commit
4. Push (perm. store): node index.js push → all commits pushed to s3
5. Pull (pull latest changes): node index.js pull
6. revert (reverts file by commit id): node index.js revert <that commit id>
